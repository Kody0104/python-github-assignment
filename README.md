# Savings Goal Tracker

This project is a simple Python program that helps you estimate how long it will take to reach a $500 savings goal based on how much money you plan to save per day. It includes user input, calculations, formatted output, and basic error handling. This project also demonstrates a professional workflow using VS Code, Git, and GitHub.

## How to Run

Make sure Python is installed. Then run:

```bash
python app.py
# or
python3 app.py
